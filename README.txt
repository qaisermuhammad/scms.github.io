Hello.

You have been sent a website for review. You do not need to understand software
engineering. Please follow these instructions exactly, step by step.


WHAT THIS WEBSITE IS
--------------------

This is a local preview of the Safi College of Medical Sciences website.

It will open on your own computer only. It is not live on the internet yet.

You are only checking the website and telling us if anything needs to be changed.


BEFORE YOU START
----------------

You need:

1. A Windows computer or laptop.
2. Internet connection for the first run.
3. Python installed on your computer.

If Python is not installed:

1. Open this website:
   https://www.python.org/downloads/

2. Download Python.

3. Install Python.

4. IMPORTANT:
   During installation, tick/check this option:

   Add python.exe to PATH

5. Finish the installation.


HOW TO OPEN THE WEBSITE
-----------------------

1. Extract/unzip the folder you received.

   Do not run it from inside the zip file.
   First extract it.

2. Open the extracted folder.

3. Find this file:

   START_WEBSITE.bat

4. Double-click:

   START_WEBSITE.bat

5. A black command window will open.

   This is normal.
   Do not close this black window.

6. Wait for the website to start.

   The first time can take a few minutes because it may install required files.

7. Your browser should open automatically.

8. If the browser does not open automatically, manually open Google Chrome,
   Microsoft Edge, or any browser and type this address:

   http://127.0.0.1:5000


VERY IMPORTANT
--------------

Keep the black command window open while checking the website.

If you close the black command window, the website will stop working.

If the website stops working, double-click START_WEBSITE.bat again.

COMMON PROBLEMS AND SOLUTIONS
-----------------------------

Problem:
The website does not open.

Solution:
Make sure the black command window is still open.
If it is closed, double-click START_WEBSITE.bat again.


Problem:
The browser says "This site cannot be reached".

Solution:
The server is not running.
Double-click START_WEBSITE.bat again and wait.


Problem:
Python is not recognized or Python is missing.

Solution:
Install Python from:
https://www.python.org/downloads/

During installation, tick/check:
Add python.exe to PATH


Problem:
The website opens but shows old content.

Solution:
Press Ctrl + F5 in the browser.
This refreshes the page fully.


Problem:
The black window shows an error about packages or Flask.

Solution:
Check your internet connection.
Then close the black window and double-click START_WEBSITE.bat again.


Problem:
The website works on the owner's computer but not on your computer.

Solution:
Make sure you extracted the complete folder.
Do not delete any files.
Do not move START_WEBSITE.bat away from the website folder.
